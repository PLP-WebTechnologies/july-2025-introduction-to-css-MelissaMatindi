Overview

This page demonstrates core CSS fundamentals: selectors, typography, backgrounds, and the Box Model (content, padding, border, margin). Use this as a lab — change values in styles.css and watch layout behavior adapt.

Cards & Visual Spacing

Cards below use consistent padding, margin and border to create a neat visual rhythm.

Padding (inside)

Padding increases the white space inside the border — it pushes content inward.

This card uses padding: 1.25rem;.

Border (edge)

Borders define the visible edge of an element and are included in the Box Model calculations.

This card uses border: 6px solid var(--accent) for emphasis.

Margin (outside)

Margins create space outside an element and separate elements from one another.

This card has a bottom margin to separate sections in flow.
